Company Structure 1.1

Esta aplicación desarrollada por EduWeb Group, que nos permite anidar compañias dentro de compañias padre. El módulo se paga una única vez por versión de Odoo. Si tiene instalada la versión Odoo 14 y actualiza a Odoo 15 habrá que adquirir el módulo para la versión de Odoo 15.
Está probado en la versión Enterprise (Odoo.sh y On premise) y Community. Recuerde que los módulos de terceros no funciona en la versión SaaS (Odoo Cloud Hosting).
Mediante esté modulo, puede gestionar su empresa de forma piramidal teniendo empresas padres que podrán ver toda la información de empresas hijas.
Desde contactos podrá visualizar la pestaña "Child Companies" y acceder a toda la información de las mismas.
Este módulo es compatible con todos los módulos de Odoo.
Por favor, informe de cualquier error o comportamiento inesperado a support@eduwebgroup.com
